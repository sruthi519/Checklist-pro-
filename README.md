# Checklist-pro-
CheckList Pro is a simple To-Do List Management System developed in C using a Singly Linked List. It allows users to add, view, search, and delete tasks while organizing them with categories, priorities, and reminders. The project demonstrates the implementation of dynamic data structures and basic task management functionalities in C.
Features
Add new tasks
View all tasks
Search tasks by title
Delete tasks by category
Organize tasks with categories and priorities
Set reminders for tasks
Dynamic memory allocation using linked lists
Technologies Used
C Programming Language
Singly Linked List Data Structure
Learning Outcomes

This project demonstrates:

Dynamic memory management
Linked list operations
Data structure implementation in C
Task management system design
